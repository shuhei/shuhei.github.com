# My Personal Website

## Setup

```sh
git clone {this repo}
git checkout source

yarn
```

## Add posts and pages

To add a post:

```sh
yarn newpost 'Hello World!'
```

To add a page:

```sh
yarn newpage 'foo'
```

## Start a dev server

```sh
yarn start
```

## Deploy

Deploy to GitHub Pages.

```sh
yarn deploy
```
