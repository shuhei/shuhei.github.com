ポインタについて勉強してみます。

h3. 変数のアドレス

pre(prettyprint). int value = 10;

のとき @&value@ で変数 @value@ のメモリ上のアドレスがわかります。

h3. ポインタ

メモリ上のアドレスを格納する変数がポインタです。

pre(prettyprint).. int value = 10;
int *ptr;
ptr = &a;

p. という感じです。

h3. ポインタの指す値

@ptr@ の指すアドレスにある変数の値をとるには @*ptr@ と書きます。

pre(prettyprint). printf("*ptr の値は %d です。", *ptr); // *ptr の値は 10 です。
