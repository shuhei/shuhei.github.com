irb で下のような関数を書いてから、ひたすら 0 と 1 を打ち込み・・・。

class String
  def bin
    Integer('0b' + self)
  end
end

def decode(ascii)
  0.step(ascii.size - 1, 8) do |num|
    puts ascii[num..(num + 7)].bin.chr
  end
end

作る方は簡単でしょうが、読む方は大変です。
